# React + Vite Frontend UI and State Management


Goal:
Build the SPA dashboard UI in React + Vite, implementing all four UI states and static API contracts.


Instructions:
Scaffold a Vite React app. Implement the Dashboard view with header, status bar, and main content area. Use mock/fake API responses matching the backend contract for initial development. Implement explicit UI states: Loading, Empty, Error (with Retry), and Success. Prepare API service layer for later backend integration.


Dependencies:
none
