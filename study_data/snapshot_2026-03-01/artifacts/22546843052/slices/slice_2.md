# React + Vite Frontend Dashboard


Goal:
Build a single-page React app that fetches and displays backend data with state handling.


Instructions:
Set up a Vite React (TypeScript) project. Implement a Dashboard view that fetches /info, /healthz, and /data from the backend. Display the app name, system version, a status bar (green for ok, red for error), and a grid/list of items from /data. Handle and visually distinguish Loading, Empty, Error (with Retry), and Success states as specified.


Dependencies:
slice_1
